
# Blueprint

## Overview

This project is a Next.js application with internationalization (i18n) support, a modern design, and a comprehensive set of features.

## Project Structure

*   `/app`: App Router directory.
    *   `/`: Root layout and page.
    *   `/about`: About page.
    *   `/contact`: Contact page.
    *   `/portfolio`: Portfolio page.
    *   `/services`: Services page.
    *   `/templates`: Templates page.
*   `/components`: Reusable React components.
    *   `Header.tsx`: Site header.
    *   `Footer.tsx`: Site footer.
    *   `LanguageSwitcher.tsx`: Language switcher component.
*   `/dictionaries`: Dictionary files for i18n.
*   `/i18n-config.ts`: Configuration for i18n.
*   `/middleware.ts`: Middleware for i18n routing.
*   `/public`: Static assets.

## Design and Features

*   **Internationalization (i18n):** The application supports multiple languages (English and Russian) using dictionary-based translation.
*   **Modern Design:** The application features a modern and visually appealing design with a focus on user experience.
*   **Responsive Layout:** The application is fully responsive and works seamlessly on both desktop and mobile devices.
*   **Component-Based Architecture:** The application is built using a component-based architecture, which promotes code reusability and maintainability.
